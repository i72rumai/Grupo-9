**HISTORIA DE USUARIO**

Función: Modificar_integrantes.

**(ANVERSO)**

Como usuario quiero poder añadir y borrar integrantes de los grupos deseados.

**(REVERSO)**

- Quiero modificar los integrantes de un grupo.
- El programa hara una llamada a Buscar_alumnos para comprobar si el alumno que
  desea modificar existe en la base de datos.
- El sistema modifica el alumno deseado.
